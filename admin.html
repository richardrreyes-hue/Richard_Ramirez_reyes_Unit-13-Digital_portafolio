CPU Instruction Flow Steps

Processing steps during the foundational Fetch-Decode-Execute CPU instruction cycle:

  1. Fetch: Memory address values register inside the Program Counter (PC), transferring payload components via internal data buses.
  2. Decode: The Control Unit (CU) interprets machine operation codes inside Instruction Registers.
  3. Execute: Calculated outputs update primary ALU and Accumulator registers.

Unit 21: Cryptography

Cryptographic Systems & Substitution Algorithms

Understanding security barriers, symmetric keys, and active cipher shift algorithms.

The Caesar Cipher Algorithm

The Caesar Cipher is a classic symmetric substitution algorithm. Each letter in the plaintext message is shifted by a constant key value ($K$) down the alphabet grid.

E_k(x) = (x + k) \pmod{26}

Where $x$ represents the position of the plaintext character and $k$ represents the shift key value parameter.

1. Purpose of Webpages and Met Metrics

Each simulated webpage within this digital portfolio satisfies a distinct grading requirement:

  • Homepage: Unifies systemic portfolios under a standard high-contrast accessibility context.
  • Unit 1 & 2: Evaluates hardware layouts, bus networks, and operational SQL constraints via live play sands.
  • Unit 6 & 7: Focuses on wireframe plans, column measurements, and mobile flow blueprints.
  • Unit 12: Highlights software algorithm logic and formal functional tests.

2. Navigation Framework & Aesthetic Choices

Colors have been limited to neutral greys, black, and whites to enforce absolute readability and focus strictly on content structure (WCAG AAA compliant). The navigation employs high contrast active state buttons inside a simulated desktop chrome. This dual-path layout allows users to easily jump from the home dashboard directly to localized unit pages, keeping navigation paths clear.

3. Meeting Technical and User Needs

This system meets college requirements by organizing all files within logically structured assets folders. Interactive elements, including form inputs, slider mechanisms, toggle triggers, and SQL code consoles, provide instant feedback notifications to replace intrusive browser alert overlays, ensuring a smooth and responsive user experience.
